# Spec F-0001: Antigravity UI Automation Reference Guide

## Overview
This project aims to create a comprehensive reference guide for automating UI tests using Antigravity agents. It will include templates for Playwright and Selenium, and a wizard to help users set up their test suites.

## Requirements
1. CLI command `antigravity-ui init` to bootstrap a test project.
2. Templates for Playwright (TypeScript) and Selenium (Python).
3. Documentation on how to use Antigravity agents for UI testing.
4. Integration with existing Antigravity workflows.
